# cursoTypeScript-Cod3r
Curso Entendendo TypeScript

## Iniciando - Montando o ambiente

### Instalando o typescript
$ npm i -g typescript

$ tsc -v // Mostra a versao do TypeScript

### Inicializando um diretorio/projeto para trabalhar com TS
$ tsc --init

### Compilar 
$ tsc <arquivo.ts>
